# Dashboard QA 테스트케이스

## 개요
다음 대시보드 기능에 대한 QA 테스트케이스입니다:
- 대시보드 메인 페이지 (`/dashboard`)
- 설정 페이지 (`/settings`)
- 사이드바 네비게이션
- 헤더 (다크모드 토글)

**작성일**: 2026-04-19  
**버전**: 1.0

## 테스트 실행 가이드

### 전제 조건
```bash
npm install
npm run dev
```
- 서버 실행: `http://localhost:3000`

### 테스트 시작
1. `http://localhost:3000/dashboard` 접속
2. TC-01부터 순차적으로 테스트 실행
3. 각 테스트에서 기대 결과와 실제 결과 비교
4. 불일치 시 버그 보고

### 테스트 환경
- **브라우저**: Chrome/Safari 최신 버전
- **OS**: macOS, Windows, Linux
- **화면 크기**: Mobile (390px), Tablet (820px), Desktop (1920px)
